# Pareto Optimization Trading System with 3D Visualization

## What is this project about?
This project is a Python-based trading system that uses multi-objective optimization to evaluate trading strategies and visualize the Pareto front in 3D. It generates synthetic (or optionally real) price data, applies various moving average strategies, and computes key metrics: return, risk (variance), and drawdown. The system identifies Pareto optimal strategies and visualizes them in a 3D scatter plot, highlighting the Pareto front.

## How to fork this project
1. Click the "Fork" button on the top right of the repository page on GitHub.
2. Clone your forked repository to your local machine:
   ```bash
   git clone https://github.com/YOUR_USERNAME/Pareto-Optimization-Trader.git
   ```
3. Install the required Python packages:
   ```bash
   pip install numpy pandas matplotlib scipy
   ```
4. Run the main script:
   ```bash
   python pareto_trader.py
   ```
